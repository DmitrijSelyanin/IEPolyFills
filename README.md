this is some open-source polyfills to implement modern web standards in Internet Explorer. let.js polyfills for let by replacing it with var. this polyfill requires ES5 and later support, so IE10 or IE11 work fine with it but IE9 and older don't. const.js polyfills for const by replacing it with var. this polyfill requires ES5 and later support, so IE10 or IE11 work fine with it but IE9 and older don't.
